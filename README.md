# ui.abdulrahmanasif.dev

A free, open-source landing page template library built with Next.js 16, React 19, and Tailwind v4. Browse templates, preview them live, and install any template into your project with a single shadcn command.

## Templates

| Name | Description |
|------|-------------|
| **Nova** | Premium SaaS landing page — hero, features, pricing, FAQ |
| **Popcorn** | Clean landing page with coverage map, features, and testimonials |
| **Orbit** | Modern SaaS page with global reach visualization |
| **Glaze** | Dark-themed page with glassmorphism effects |
| **Nexora** | AI image generation page — hero, how-it-works, pricing |
| **Lumina** | White + indigo SaaS automation page with frosted-glass dashboard preview |
| **Nexto** | Studio/agency page — services grid, showcase, process steps |
| **Xero** | Dark SaaS page for data/encryption with animated beam hero |
| **Aurora** | Premium dark-mode page with video hero, marquee, and feature grid |

## Install a template

```bash
npx shadcn@latest add https://ui.abdulrahmanasif.dev/r/<slug>.json
```

Replace `<slug>` with any template name (e.g. `nova`, `glaze`, `aurora`).

## Run locally

```bash
bun install
bun dev        # http://localhost:3000
```

## Stack

- **Next.js 16** (App Router, two root layouts)
- **React 19**
- **Tailwind v4** (`@theme inline` blocks, no `tailwind.config.ts`)
- **shadcn/ui** (`base-nova` style)
- **motion** (framer-motion v12+)
- **next-themes**
